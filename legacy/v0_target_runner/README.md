# Legacy V0 Target Runner

This folder contains the older target-by-target ReconForge runner.

It is preserved for reference only while ReconForge V1 moves to a program-level workflow.

Current V1 entrypoint:

```bash
python3 reconforge.py run --program PROGRAM_NAME --scope-csv SCOPE_CSV --profile standard
```

Do not use this legacy runner for new program-level Bug Bounty reconnaissance.
